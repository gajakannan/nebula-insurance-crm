**Status:** Draft
**Priority:** Critical

## Overview

Define and implement the foundation for product-specific insurance attributes through governed JSON Schemas, schema-pinned persistence, shared validation, and dynamic forms. The first implementation should prove a narrow product pilot before F0019 adds quoting and proposal fields that would otherwise become fixed-column and custom-form debt.
