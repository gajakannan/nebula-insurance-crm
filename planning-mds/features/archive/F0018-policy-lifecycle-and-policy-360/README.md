# F0018 — Policy Lifecycle & Policy 360

**Status:** Done
**Archived:** 2026-04-22
**Priority:** Critical
**Phase:** CRM Release MVP

## Overview

Introduce Policy as a first-class insurance record with a dedicated Policy 360 workspace, explicit lifecycle states (Pending / Issued / Expired / Cancelled with LOB-configurable reinstatement), immutable version snapshots, structured coverage lines, endorsement events, predecessor/successor renewal linkage, and full audit timeline — so Nebula is credible as a commercial P&C CRM system of record.
